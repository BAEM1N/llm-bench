## 해석 주의사항

| # | 항목 | 내용 |
|---|------|------|
| 1 | **모델 가중치 소스 차이** | MLX(mlx-community 4-bit) ≠ llama.cpp/Ollama(unsloth GGUF). 엔진+가중치 패키지 통합 비교 |
| 2 | **Ollama KV 캐시 pre-allocation** | num_ctx=262144 설정으로 30~40GB KV 캐시 사전 할당 → 메모리 비교에서 구조적 불리 |
| 3 | **Ollama gen 트랙 TTFT 무효** | TTFT가 max_tokens에 비례 증가 → Section 4(gen 트랙 Prefill TPS) 수치 신뢰 불가, Section 5(Prefill 트랙)만 유효 |
| 4 | **Flash Attention 비대칭** | llama.cpp on / MLX 미적용 / Ollama 추정 — prefill 비교에 알고리즘 수준 차이 포함 |
| 5 | **122B Q8_0 미측정** | 128GB 한계 초과 — 전 백엔드 적재 불가 |
